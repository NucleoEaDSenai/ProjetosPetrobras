🔥 Configurar Firebase
Cole a URL do seu Realtime Database abaixo.
Isso só precisa ser feito uma vez — depois fica salvo.
Como obter sua URL:
1. Acesse console.firebase.google.com
2. Crie um projeto (grátis) → clique em Realtime Database
3. Clique em Criar banco de dados → escolha Iniciar no modo de teste
4. Copie a URL que aparece no topo, ex:
https://meu-projeto-12345-default-rtdb.firebaseio.com
